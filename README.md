# JX1000 Device APIs

This repository contains a comprehensive set of APIs and tools for interacting with the **JX1000 device**. It includes:

- **C# API** - for integration into .NET applications.
- **Python API** - for scripting, automation, and rapid prototyping.
- **Custom Modbus API** - for direct, low-level register communication with the device.

These APIs provide functionality for:

- Reading and writing device registers
- Initializing and configuring the JX1000
- Simplifying integration into custom software projects

This project is designed to make it easy to communicate with the JX1000 using the language and interface that best fits the workflow.

---

## Features

- Cross-language support: C# and Python
- Direct Modbus communication for low-level device control
- Simple read/write register functions
- Device initialization and configuration utilities
- Example scripts for quick testing and integration
